# Pet-ProjectFront

1)ProjectGame
Простой Интерфейс в котором есть: Изменение темы(Тёмная, Светлая, Цветные темы на выбор(13шт)).
Элемент с анимацией в центре, можно менять его оттенки ободки: Голубые, Красные, Зелёные.



2)ProjectFront
В этом пет-проекте есть 3 элемента с анимацией которую можно включать и выключать.
Цвета Первого элемента имеют голубые оттенки, Второго - зелёные, у Третьего элемента Радужные цвета(RGB).
